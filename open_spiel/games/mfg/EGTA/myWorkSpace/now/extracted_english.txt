Game Model Learning for Mean Field Games
Paper #86
Abstract. We present an approach to learning models for mean
field games (MFGs) from simulation data. To model an MFG, the
central object is to capture the result of a strategy played by a rep-
resentative player in the context of the distribution over the infinite
population of other players (i.e., the mean field). We thus define the
problem of learning a utility function that maps from a restricted set
of strategies and possible mean fields to utility values. Starting from
the time-dependent structure typically employed in defining MFGs,
explicitly encoding the strategies and mean fields would make for an
impractically complex regression setup. To handle this issue, we de-
velop a coarse coding scheme that abstracts away the time-dependent
complexity and dramatically simplifies the input representation. To
endow the learned utility function with the power of generalization
across the space of mean fields, one key step is to uniformly sam-
ple strategies that induce mean fields in the restricted strategy space,
which is typically high-dimensional. To handle the sampling in high-
dimensional strategy spaces, we employ a combination of two sam-
pling schemes: grid sampling and sampling from Dirichlet distribu-
tions with varying concentration parameters. Through experiments
on some standard MFGs, we show that our methods achieve effective
generalization and the learned game models can successfully support
game-theoretic analysis.1
1 Introduction2
Mean field games (MFGs) [12, 8] describe systems with a conceptu-3
ally infinite number of interacting strategic players. Players within4
a population are treated atomistically and identically, so that the5
state of the system can be captured by a probability distribution over6
player states (the “mean field”). Analysis in MFGs can be reduced7
to the characterization of the optimal behavior of a single representa-8
tive player in its interactions with the full population, as represented9
by the mean field. The MFG framework can support game-theoretic10
analysis that would be intractable for a standard corresponding model11
of a game among a large but finite number of players.12
Despite MFGs provide a simplified framework, game-theoretic13
analysis is still hindered by the large strategy spaces in MFGs. To14
handle the large strategy spaces, one popular methodology that has15
been widely applied in finite games is to construct a game model that16
approximates the true game and analyze the true game by reasoning17
about the game model [24, 23, 27]. A game model involves only a18
particular set of strategies and a utility function induced from sample19
utilities over combinations of the set of strategies. Due to the restric-20
tion on the strategy space, a game model is considered as an succinct21
description of the true game and can be analyzed readily.22
Although being effective in finite games, the methodology cannot23
be directly applied to MFGs on account of lack of a well-defined24
game model. To define a game model, a key step is to induce a utility25
function over the restricted set of strategies. As a result of the non-26
linearity of the utility function of MFGs in the mean field [18, 26]27
and the continuity of the space of mean fields, representing a utility 28
function explicitly over mean fields derived by any strategy mixture 29
becomes completely infeasible. Without a proper representation of 30
the utility function, a game model currently has not been well-defined 31
for MFGs in prior work and thus the methodology for game analysis 32
fails to apply. 33
In this work, we fill in this gap and propose to define a game model 34
for MFGs by learning a utility function over a restricted set of strate- 35
gies and distributions derived by these strategies. We study the gen- 36
eral setup of MFGs where strategies and distributions are both time- 37
dependent (i.e., non-stationary), and hence encoding them explicitly 38
as inputs to a learner entails impractically high dimensionality. To 39
handle the time-dependency, we propose a coding scheme and learn 40
a utility function that takes as inputs sufficiently-statistical represen- 41
tations of strategies and distributions, and outputs a utility. With our 42
method, time-dependent strategies and distributions are no longer ex- 43
plicitly encoded as inputs as for the true utility function and hence 44
our method circumvents the representation complexity induced by 45
the time-dependency. We refer to this method ascoarse coding. 46
To learn an effective utility function for MFGs, it is important to 47
endow the learned utility function with the ability of generalization 48
across the space of strategies that induce mean fields. To reach this 49
goal, the training data set is required to include uniformly-sampled 50
mixed strategies in the restricted strategy space. For a large MFG, an 51
effective game model typically involves dozens of strategically sig- 52
nificant strategies, which creates a high-dimensional strategy space. 53
To obtain samples in such high-dimensional spaces, we propose a 54
combination of two sampling schemes: grid sampling and sampling 55
from the Dirichlet distribution with varying concentration parame- 56
ters. By combining coarse coding with our data sampling methods, 57
we demonstrate that our approach can successfully achieve effective 58
generalization and accurate predictions on utilities. We also show 59
that the learned utility function can support game-theoretic analysis, 60
that is, both fictitious play (FP) [21] and replicator dynamics (RD) 61
[26] empirically converge to NE with the learned utility function. 62
The main contribution of this study is a learning approach for 63
constructing game models for MFGs. We propose a coarse coding 64
scheme that successfully handles the high-dimensional representa- 65
tion of the time-dependent strategies and distributions. We propose 66
a data sampling scheme to enable generalization of a learned model, 67
which combines grid sampling and sampling from the Dirichlet dis- 68
tribution with varying parameters. Through experiments on some 69
standard MFGs, we demonstrate the learned model can effectively 70
predict utilities and achieves efficacious generalization. Finally, we 71
show that our learned model can successfully support game-theoretic 72
analysis. 73

2 Related Work74
2.1 Game Model Learning in Finite Games75
In essence, game model learning is a form of regression that learns76
the utility function over a restricted set of strategies. To the best of77
our knowledge, our work is the first study of game model learning78
in MFGs. We briefly survey prior work on game model learning for79
finite games.80
V orobeychik et al. [25] first introduced the concept of learning81
normal-form game models as utility function regression from sim-82
ulation data sampled over continuous strategy spaces. They demon-83
strated the approach using single-parameter strategy representation.84
Ficici et al. [6] clustered a large number of players into two roles85
based on data consisting of strategy profiles and utilities. Then re-86
gression of the utility function was applied for each role.87
Wiedenbeck et al. [28] deployed Gaussian process regression88
(GPR) to learn the utility function of large symmetric games. The89
regressor takes as input a pure strategy profile and outputs a utility90
vector. In symmetric games, since the utility function depends on91
how many players choose each strategy (and not which players), the92
input vector to the regressor can be represented by a non-negative93
tally vector with one dimension per strategy. Once the utility func-94
tion for pure strategy profiles is learned, the extension to mixed strat-95
egy profiles can be achieved by taking the expectation. The authors96
further investigated the use of neural networks for the regression.97
Sokota et al. [22] extended game model learning to role-symmetric98
games by regressing the deviation payoff function rather than the99
payoff function. Deviation payoffs can be directly used by algorithms100
such as replicator dynamics, saving the effort to derive deviation pay-101
offs first from the payoff function. This approach has also been de-102
ployed by Li and Wellman [15] to learn NE in Bayesian games.103
There has also been some work on learning game models on some104
succinct descriptions of games. For example, Duong et al. [3] and105
Fearnley et al. [5] learned graphical game models [9] from utility106
data. Li and Wellman [14] combined structure learning and payoff107
regression to induce tractable game models with many players.108
For MFGs, there is a considerable literature on learning equilibra109
[2, 7, 21, 17, 18, 13, 26]. Despite we show that the learned utility110
function can facilitate equilibrium search in our experiments, our ap-111
proach differs from these works in nature since our object is to learn112
the utility function of MFGs rather than presenting an equilibrium113
learning algorithm. Therefore, they are not fair baselines for our ap-114
proach.115
2.2 Learning with Game Models in MFGs116
Muller et al. [19] adapted Policy Space Response Oracles (PSRO)117
framework [10] to mean-field games (MFGs), which essentially con-118
ducts game-theoretic analysis through analyzing game models in119
which the set of strategies are constructed by reinforcement learning.120
However, the game models used in their work are not well-defined121
because the utility function for MFGs is not generally linear in the122
mean field and thus the empirical MFG model cannot be represented123
explicitly. Our approach provides a practical way to construct game124
models through learning. Using a learned game model has potential125
to dramatically decrease the cost of analyzing an empirical game in126
two ways. First, a learned game model can avoid querying the same127
or similar profile multiple times. Second, a learned game model can128
accelerate learning dynamics on empirical games by circumventing129
the induction of distributions.130

4 Methods167
4.1 Time-dependent Strategies and Distributions168
The primary goal of game model learning is to learn the utility func-169
tion ˜u(σ, µ) over a restricted set of strategies. Since˜u(σ, µ) is an ex-170
pectation of ˜u(s, µ) over s ∈ Λ (this follows Equation 1), it is suffi-171
cient to learn the utility function˜u(s, µ) on pure strategies. In MFGs,172
a strategy s and distributions µ are generally both time-dependent173
(i.e., s = ( st)t∈[0,T −1] and µ = ( µt)t∈[0,T]). Explicitly encoding174
them as inputs to a learner would result in a high-dimensional fea-175
ture vector and entail an impractically complex regression setup. To176
handle this issue, we propose a coarse coding scheme.177
4.2 Coarse Coding178
In finite games, a utility function can be treated as a black box (il-179
lustrated in Figure 1) and abstracts away the details of strategies as180
well as the mechanism of the utility computation. A black-box util-181
ity function takes a simple representation of a strategy profile I(s)182
(i.e., a vector of strategy indices one per player) as input and outputs183
utility samples.184
Figure 1. A black-box utility function.
We are inspired by this abstraction and handle the time-185
dependency by learning a black-box version of the true utility func-186
tion ˜u. Mathematically, consider a restricted strategy set Λ ⊆ S. Let187
I : Λ → Z+ be an indexing function that assigns a unique index to188
each strategy s ∈ Λ. Let σ be the mixed strategy that induces the189
distributions µσ. Since the distribution induction function Φ is de-190
terministic, the distributions µσ are uniquely determined by a fixed191
initial distribution µ0 ∈ ∆(X) and a strategy σ.1 Rather than learn-192
ing ˜u(s, µσ) with time-dependent inputs, we learn a black-box util-193
ity function ˆu : I(Λ) × ∆(Λ) → R with equivalent representations194
I(s) and σ of s and µσ, respectively. We refer to this representation195
as coarse coding.196
Our object is to predict the true utility ˜u(s, µσ) by ˆu(I(s), σ) and197
thus minimizing the mean square loss E[(˜u(s, µσ) − ˆu(I(s), σ))2].198
Our regression is based on neural networks and the structure of the199
neural networks is shown in Figure 2. In practice, I(s) can be any200
representation of categorical inputs (e.g., one-hot encoding) and σ is201
a vector of strategy probabilities.202
There are two main benefits of coarse coding. First, the coarse203
coding scheme only requires a simple network structure with a one-204
hot encoded strategy and a vector representation of a mixed strategy.205
So the implementation of coarse coding is straightforward.206
Second, the coarse coding scheme can dramatically simplify the207
execution of equilibrium search algorithms for MFGs. Consider FP208
as an example. FP for MFGs can be summarized as iterating three209

which distribution induction is implicitly learned from data. Based 227
on these benefits, FP can be dramatically accelerated given a coarse- 228
coded utility model. 229
4.3 Data Sampling 230
For regression in our case, a data point constitutes an index of a 231
pure strategy I(s), a mixed strategy σ, and a true utility ˜u(s, µσ). 232
To collect these data points, the basic requirement is that the sam- 233
pled mixed strategies σ’s should uniformly distribute in the restricted 234
strategy space so as to endow the learner with the ability of general- 235

a game model typically contains dozens of strategies, which makes 237

4.4 Approximating Nash Equilibrium273

evaluation, we consider two learning dynamics FP and RD, and im-276
plement them using our learned utility function. For each learning277
dynamic, we measure the regret of intermediate strategies generated278
by the dynamic with the true utility function and the learned one,279
respectively. If the regrets given by the true utility function and the280
learned one are close to each other and both converge to 0 (i.e., ap-281

beach bar problems with different complexity. In the 1-dimensional

6 Conclusion364
In this study, we develop a game model learning approach for MFGs.365
We introduce a coarse coding scheme to handle the high-dimensional366
inputs in the utility function of MFGs and a data sampling scheme367
for sampling in high-dimensional strategy spaces. We show that the368
learning curves generated with our learned utility function almost369
coincide with those given by the true utility function, which demon-370
strates our learned utility function is accurate and can support game- 371
theoretic analysis in MFGs. By combining the population, the re- 372
stricted set of strategies, and a learned utility function, a game model 373
for MFGs is well-defined. 374
In our game model learning approach, each strategy is represented 375
by an index. Although an index coding is beneficial to handling time- 376
dependency, it does not provide any information about the similar- 377
ity among strategies. Future research will include replacing the in- 378
dex representation with the embedding (i.e., a summary) of param- 379
eterized strategies, enabling the generalization of the learned game 380
model to new strategies. 381
